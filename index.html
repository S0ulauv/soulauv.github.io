面向对象编程

如何理解？

在学习的过程中，偶然间看到知乎上一个转载片段讲的很生动形象，
面向过程编程就像是做一盘蛋炒饭，需要把每一个步骤单独开来，比如煮米饭、炒鸡蛋、混合…… 最后的步骤是把炒好的鸡蛋和大米混合在一起。
而面向对象编程( Object Oriented Programming )则不同，类比于盖浇饭，分别做好饭和盖菜，最后想要什么样的盖浇饭就在米饭上面盖什么样的盖菜。
